# file-upload

[Hongbusi/file-upload](https://github.com/Hongbusi/file-upload) 学习版本，基于 [Hongbusi/vitemplate](https://github.com/Hongbusi/vitemplate) 快速搭建。

## 阅读指南

强烈推荐按照下面的顺序阅读学习。

- 选择文件
- 文件抽样哈希计算

## client

``` bash
cd client

pnpm install

pnpm dev
```

## server

``` bash
cd server

pnpm

pnpm dev
```
